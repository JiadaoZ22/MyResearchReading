# Reference Guide: Where to Find Data in Figures and Tables

## Important Note

Many specific values mentioned in the document may be extractable from figures and tables, even if they are not explicitly stated in the main text. This guide lists where to find specific information.

## Key Figures and Tables

### **Fig. 2**: CH normative references of brain structural measures
- **Contains**: Normative curves for all global and regional measures
- **Extractable data**: 
  - Approximate volume/thickness values by age
  - Percentile curves (5th, 50th, 95th percentiles)
  - Peak ages and trajectory patterns
- **Use for**: Getting approximate normative values for specific ages

### **Fig. 3**: Deviations of global and regional structural measures
- **Contains**: Deviation score distributions and maps for all diseases
- **Extractable data**:
  - Median deviation scores for each ROI
  - Regional deviation patterns
  - Disease-specific atrophy patterns
- **Use for**: Understanding which ROIs show most deviation in AD

### **Fig. 4a**: Classification performance
- **Contains**: AUC values and DPS distributions for all diseases
- **Extractable data**:
  - AUC values with confidence intervals
  - DPS median and IQR values
- **Use for**: Classification performance metrics

### **Extended Data Fig. 9a**: Effect sizes (Cohen's d)
- **Contains**: Comprehensive Cohen's d values for all ROIs across all diseases
- **Extractable data**:
  - Cohen's d for every ROI in every disease
  - Effect size maps
- **Use for**: Getting complete effect size information for all ROIs

### **Extended Data Fig. 9e**: Contributing features
- **Contains**: Feature importance/weights for classification models
- **Extractable data**:
  - Feature weights or importance scores
  - Top contributing ROIs for each disease
- **Use for**: Understanding which features contribute most to AD classification

### **Extended Data Fig. 9f**: Confusion matrices
- **Contains**: Classification confusion matrices for all diseases
- **Extractable data**:
  - True positives, false positives, true negatives, false negatives
  - Can calculate: Sensitivity, Specificity, PPV, NPV
- **Use for**: Getting sensitivity/specificity values

### **Extended Data Fig. 1**: Sample distribution
- **Contains**: Age distribution and sample sizes
- **Extractable data**:
  - Sample sizes by age group
  - Geographic distribution
- **Use for**: Understanding sample composition

### **Table 1**: Demographic information
- **Contains**: Demographic and clinical information
- **Extractable data**:
  - Age, sex distribution
  - Clinical scores
- **Use for**: Understanding study population

## Recommendations

1. **For specific values**: Refer to the appropriate figure/table directly
2. **For normative values**: Use Fig. 2 or contact authors for GAMLSS models
3. **For effect sizes**: Use Extended Data Fig. 9a for comprehensive data
4. **For classification metrics**: Use Extended Data Fig. 9f for confusion matrices
5. **For feature importance**: Use Extended Data Fig. 9e

## Limitations of Text Extraction

PDF text extraction may not capture:
- Values in tables (especially complex tables)
- Values in figures (requires visual inspection)
- Values in Extended Data figures
- Values that are part of images/charts

Therefore, when the main text says "see Extended Data Fig. X" or "shown in Fig. Y", those sources should be consulted directly for the specific values.
